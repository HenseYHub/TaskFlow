# TaskFlow

TaskFlow is a simple iOS productivity app built with SwiftUI.

## Features

- Create and edit tasks
- Daily timeline view
- Focus timer
- Local notifications
- Completed tasks tracking
- Profile and settings
- Multi-language support (EN / DE / UA)

## Tech Stack

- SwiftUI
- MVVM
- Firebase Auth & Firestore
- UserNotifications
- Local JSON storage
